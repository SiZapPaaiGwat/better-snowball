# better-snowball

雪球助手，愉快地在雪球、理性仁等网站之间玩耍（跳转）。

[greasyfork](https://greasyfork.org/zh-CN/scripts/36582-%E9%9B%AA%E7%90%83%E5%8A%A9%E6%89%8B)

目前提供的功能有：

- 雪球主页自选股跳转到理性仁股票主页
- 雪球个股主页跳转到理性仁、同花顺F10、股票报告网、智通财经（仅港股）
- 理性仁个股主页跳转到雪球个股主页
